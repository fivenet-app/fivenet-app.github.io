Wählen Sie den Installationsweg, der zu Ihrer Umgebung passt, egal ob Sie verwaltetes Hosting mit FiveNet Cloud oder eine selbstgehostete Bereitstellung mit Docker oder Kubernetes möchten.

::callout
---
color: secondary
icon: i-mdi-information
---
**Wichtig**: Die selbstgehostete Installation von FiveNet erfordert fortgeschrittene Kenntnisse im Umgang mit Docker oder Kubernetes.<br>
Für professionellen Support, eine vereinfachte Einrichtung oder verwaltetes Hosting empfehlen wir [FiveNet Cloud](cloud).
Bitte stellen Sie vor der Installation sicher, dass Sie ausreichende Erfahrung mit diesen Technologien besitzen. Für Installation und Konfiguration steht ausschließlich Community-basierter Support zur Verfügung.
::

## Cloud-gehostet

Für diejenigen, die eine unkomplizierte Erfahrung bevorzugen, bietet die FiveNet-Cloud-Version eine optimierte Möglichkeit, FiveNet für Ihren Rollenspielserver auszuführen. Erfahren Sie mehr über das Angebot auf der [Cloud-Info-Seite](cloud).
Für alle, die eine unkomplizierte Erfahrung bevorzugen, bietet FiveNet Cloud eine vereinfachte Möglichkeit, FiveNet für Ihren Rollenspielserver zu betreiben. Erfahren Sie mehr auf der [Cloud-Infoseite](cloud).

::card
---
title: FiveNet Cloud
icon: i-mdi-cloud
to: /getting-started/installation/cloud
---
Starten Sie mit FiveNet Cloud für eine vollständig verwaltete Lösung, die manuelle Einrichtung und Wartung überflüssig macht. Ideal für alle, die sich auf ihren Rollenspielserver konzentrieren möchten, ohne sich um die Infrastruktur kümmern zu müssen.
::
