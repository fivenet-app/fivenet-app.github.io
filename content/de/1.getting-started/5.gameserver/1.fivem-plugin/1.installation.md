Installieren Sie das FiveNet-FiveM-Plugin, um Ihren Gameserver mit Ihrer FiveNet-Instanz zu verbinden.

## Installationsschritte

Um FiveNet mit Ihrem FiveM-Gameserver zu integrieren:

1. **Plugin herunterladen**
    - Laden Sie den [FiveM-Plugin-Code](https://github.com/fivenet-app/plugins/tree/main/fivem/fivenet) von GitHub herunter.
    - Verwenden Sie die neueste Version, um Kompatibilitätsprobleme mit neueren FiveNet-Releases zu vermeiden.

2. **Plugin installieren**
    - Legen Sie die Plugin-Dateien im `resources`-Verzeichnis Ihres FiveM-Servers ab.
    - Stellen Sie sicher, dass der Ressourcenordner `fivenet` heißt, sofern Sie Ihre Serverkonfiguration nicht bewusst auf einen anderen Namen anpassen.

::callout
**Hinweis:** Aktuell müssen Sie die **`yarn`-Builder-Systemressource** patchen, damit das FiveNet-Plugin funktioniert. Dies ist eine temporäre Übergangslösung, bis FiveM offiziell auf eine neuere Node.js-Version umstellt.

Aktualisieren Sie in `yarn/yarn_builder.js` die an `yarn install` übergebenen Argumente, sodass das Flag `--ignore-engines` enthalten ist. Dadurch kann der Plugin-Build Engine-Prüfungen umgehen, die nicht zu Ihrer FiveM-Umgebung passen.

3. **Plugin konfigurieren**
    - Öffnen Sie `config/client.lua` und `config/server.lua`.
    - Passen Sie die Werte so an, dass sie zu Ihrer FiveNet-Instanz, Ihrem Framework und Ihren Serveranforderungen passen.
    - Folgen Sie der [Plugin-Konfigurationsseite](2.configuration.md) für die wichtigsten Optionen.

4. **Event-Trigger hinzufügen**
    - Prüfen Sie Ihre vorhandenen Framework- und jobbezogenen Plugins und fügen Sie die notwendigen Event-Trigger hinzu, damit FiveNet die erforderlichen Aktivitätsdaten erfassen kann.
    - Nicht alle benötigten Events sind in Standard-ESX-basierten Ressourcen vorhanden, daher können Anpassungen nötig sein.

5. **Plugin starten**
    - Fügen Sie das Plugin Ihrer `server.cfg` hinzu:
      ```cfg
      ensure fivenet
      ```
    - Starten Sie Ihren Server neu, damit die Ressource geladen wird.

## Nach der Installation

Sobald das Plugin installiert ist:

- prüfen Sie, ob das Plugin Ihre FiveNet-Instanz erreichen kann
- verifizieren Sie, dass API-Host und Token korrekt konfiguriert sind
- testen Sie das Tablet im Spiel und prüfen Sie, ob die erwarteten Sync- und Event-Funktionen funktionieren

::callout
**Wichtig:**

- Lesen Sie immer die Datei `README.md`, die mit dem Plugin bereitgestellt wird, für detaillierte Setup-Anweisungen und Tipps zur Fehlerbehebung.
- Stellen Sie sicher, dass Ihr Server eine kompatible Version von FiveM und dem gewählten Framework ausführt, um Konflikte zu vermeiden.
::
