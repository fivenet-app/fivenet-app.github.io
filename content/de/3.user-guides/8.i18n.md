## Mitwirken

Wir freuen uns über Beiträge zur Verbesserung oder Erweiterung der Sprachunterstützung. Wenn Sie eine Sprache beitragen oder Übersetzungen korrigieren möchten, erstellen Sie ein Issue im FiveNet-GitHub-Projekt oder treten Sie direkt über [FiveNets POEditor-Projektlink](https://poeditor.com/join/project/hszo85uo3K) bei.

### So fügen Sie eine neue Sprache hinzu

1. Erstellen Sie ein Issue im FiveNet-GitHub-Projekt, um die gewünschte neue Sprache zu besprechen.
2. Fügen Sie im Issue folgende Informationen hinzu:
   - Sprachname (zum Beispiel `Französisch`)
   - Sprachcode (zum Beispiel `fr`)
   - Sprachsymbol (zum Beispiel `i-flagpack-fr`)
3. Nach Freigabe wird die Sprache in POEditor angelegt.
   - Anschließend können Sie ein POEditor-Konto erstellen und über [den Projektlink](https://poeditor.com/join/project/hszo85uo3K) beitreten.
4. Sie können dann beginnen, die Strings im POEditor-Projekt zu übersetzen.
5. Entwickler synchronisieren die Übersetzungen von POEditor regelmäßig mit dem FiveNet-Projekt.
   - Nach Synchronisierung und Release ist die neue Sprache in der Anwendung verfügbar.
   - Bei Problemen erstellen Sie bitte ein GitHub-Issue oder fragen Sie auf dem FiveNet-Discord-Server nach.
