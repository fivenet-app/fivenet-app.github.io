Stellen Sie sicher, dass Sie das Repository mit dem Flag `--recurse-submodules` klonen oder fuehren Sie `git submodule update --init --recursive` aus, nachdem Sie das Repository bereits geklont haben.

Dies ist erforderlich, damit die Livemap-Kacheln waehrend der Entwicklung bzw. beim Build verfuegbar sind.
::

## Erforderliche Werkzeuge

- **Golang** (Die Mindestversion finden Sie in der [`go.mod`-Datei](https://github.com/fivenet-app/fivenet/blob/main/go.mod))
    - Hinweise zur Installation von Go auf Ihrem System finden Sie in der [Go-Installationsanleitung](https://go.dev/doc/install).
    - Stellen Sie sicher, dass Ihr `PATH` `$GOPATH/bin` enthaelt, damit interne und automatisch installierte CLI-Tools korrekt funktionieren.
- **`pnpm`**
    - Hinweise zur Installation von `pnpm` auf Ihrem System finden Sie in der [pnpm-Installationsanleitung](https://pnpm.io/installation).
- **`buf` CLI**
    - Installationsanweisungen fuer `buf` finden Sie [hier](https://buf.build/docs/cli/installation/).
- **`protoc-gen-doc`**
    - Wird automatisch installiert, wenn Sie `make gen-proto` ausfuehren.
    - Installieren Sie mit:
      ```bash
      go install github.com/pseudomuto/protoc-gen-doc/cmd/protoc-gen-doc@latest
      ```
- **`go-licenses`**
    - Wird automatisch installiert, wenn Sie `make gen-licenses` ausfuehren.
    - Installieren Sie mit:
      ```bash
      go install github.com/google/go-licenses@latest
      ```

## Codium/VSCode-Benutzer

- Benutzerdefinierte Einstellungen fuer Go-, JS/TS- und Vue-Formatierung sowie Linting sind im Repository im `.vscode`-Ordner enthalten. Es wird empfohlen, diese Einstellungen fuer eine konsistente Entwicklungsumgebung zu verwenden.
- Zusaetzlich ist im Repository eine `.editorconfig`-Datei enthalten, um konsistente Codestile ueber verschiedene Editoren und IDEs hinweg sicherzustellen. Stellen Sie sicher, dass die EditorConfig-Unterstuetzung in Ihrem Editor aktiviert ist.
- Eine Liste empfohlener Erweiterungen fuer VSCode/Codium finden Sie in der Datei [.vscode/extensions.json](https://github.com/fivenet-app/fivenet/blob/main/.vscode/extensions.json).
