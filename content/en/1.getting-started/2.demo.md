## Best Way to Explore

The demo account includes multiple characters. For the best overview, use the **Los Santos Police Department (LSPD)** character with the `police` job.

This character has access to **superuser mode**, which lets you:

- Switch between jobs
- Explore administrative features

You can enable or disable superuser mode from the **user menu** in the left sidebar.

::callout{color="info" icon="i-mdi-lightbulb-on-outline"}
If you want to explore as much of FiveNet as possible, start with the police character and enable superuser mode.
::
