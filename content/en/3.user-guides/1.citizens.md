## Features

- Search citizens by:
  - Name
  - Date of birth
  - Phone number
  - Traffic infraction points
  - Open fines
- View a citizen profile with:
  - Personal information
  - Address
  - Phone number
  - Licenses (for example driver or firearm licenses)
  - Vehicle ownership
  - Open fines
  - Blood type (for medics)
- Manage citizen properties:
  - Wanted status
  - Traffic infraction points
  - Mug shot
  - Labels (for example `Dangerous` or `Violent`)
  - Additional custom properties
- Open related documents, such as case files.
