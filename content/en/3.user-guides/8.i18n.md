## Contributing

We welcome contributions to improve or expand language support. To add a language or fix translations, create an issue in the FiveNet GitHub project or join directly through [FiveNet's POEditor project](https://poeditor.com/join/project/hszo85uo3K).

### How to add a new language

1. Create an issue in the FiveNet GitHub project to discuss the language you want to add.
2. Include the following details in the issue:
   - Language name (for example `French`)
   - Language code (for example `fr`)
   - Language icon (for example `i-flagpack-fr`)
3. After approval, the language will be added to POEditor.
   - Then create a POEditor account and join through [the project link](https://poeditor.com/join/project/hszo85uo3K).
4. You can then start translating the strings in the POEditor project.
5. Developers sync translations from POEditor to FiveNet regularly.
   - After sync and release, the language becomes available in the app.
   - If you run into issues, create a GitHub issue or ask in the FiveNet Discord server.
