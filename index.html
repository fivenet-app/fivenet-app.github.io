What is FiveNet?

FiveNet is an open source project with several features to enhances your communities roleplay. From searching the state's citizen and vehicles database, filling documents for police investigations, court, and livemap of your colleagues and dispatches. A plugin especially for FiveM servers is available that enalbes the in-game tablet.

Features of FiveNet

Search Citizens
Search and find information about Citizens, including basic info, their licenses and related documents.
Vehicle Register
Search and find information about Vehicles.
Document Management
Search, create and manage Documents, using templates or from scratch.
Jobs
Infos about your job and employee management (e.g., conduct register, timeclock).
Calendar
A calendar for each individual and factions, with a RSVP functionality.
E-Mails
Write emails to colleagues and citizens for quick communication.
Livemap
Live position of dispatches and your colleagues.
Dispatch Center
Dispatch Center, for dispatching Dispatches to colleagues, managing units and dispatches.
Multi-Language UI
Multiple languages available for the UI.
Nuxt 3 + Nuxt UI Pro
Powered by Nuxt 3 and Nuxt UI Pro for optimal performance.
Open Source
FiveNet's code is licensed under the Apache 2.0 license.

Ready to be used In-Game

FiveNet plugins are available for FiveM servers at the moment. Using a FiveNet Cloud instance is the fastest way to get you started with the in-game tablet.
  • Your FiveNet Cloud instance will be ready in a few seconds.
  • Each FiveNet Cloud instance will be securely reachable via HTTPS and its own subdomain.
  • The plugin code is easily extendable by your developers to integrate with your servers framework and plugins (e.g., fines, location tracking).
